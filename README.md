# VSOSH Munic Jeopardy Trainer (локально)

Локальный сайт-тренажёр под муниципальный этап (ИБ): таски, подсказки/решения (3 уровня), вкладка для райтапов и сохранение прогресса.

## Запуск через Docker Compose

1. Установи Docker Desktop.
2. В этой папке выполни:
   ```bash
   docker compose up --build
   ```
3. Открой: http://localhost:8000

Прогресс хранится в Docker volume `ctf_data` (SQLite), поэтому не пропадает после перезапуска.

## WEB таски

WEB-челленджи запускаются отдельными сервисами на `localhost` (порты `8101-8108`). Кликабельные ссылки есть прямо в описании каждого WEB таска на главной странице.

## Что внутри

- `Задачи` - список тасков, ввод флага (`vsosh...`), подсветка решённых.
- `Решения` — кнопка «подсказка»: 1-я лёгкая, 2-я сильнее, 3-я — полное решение.
- `Райтапы` - твои заметки/райтап по каждому таску (сохраняется).

## Сброс прогресса

Если нужно начать заново, удали volume с базой:

```bash
docker compose down -v
```
